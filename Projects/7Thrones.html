7 Thrones

Multiplayer Game

Unity - HTTP/REST RestClient - Socket.IO

7 Thrones is a multiplayer mobile game, where the player can buy game coins to play and bet money, the game is simple the player chooses a door that has a cost, then chooses a throne where he wants to play, then it waits for other 6 players to join the game, the game starts and the player has to choose card 1 or 2, then the game chooses a random winner based on the card and throne it chooses.
The game is created with unity using RestClient based on promises for the HTTP requests to the server (GET and POST), we used Socket.io in unity and in the server created with node.js ,for real time comunication with server and client.
