Virtual Reality - Augmented Reality and Normal Mode

Unity - ARcore - ARkit - Cardboard

Personal project to practice some skills and trying to make it look very real.
I've used google cardboard for the VR mobile experience using eyes where you can click and teleport to them. For the AR I used the same apartment but adapt it for AR using Unity AR Foundation for ARkit and ARcore to get that working on IOS and Android at the same time. The normal mode is like a game where you have a left joystick to move and touching the screen to move the camera.
