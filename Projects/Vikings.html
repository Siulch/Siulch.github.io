Find vikings things

Virtual Reality

Unity - Cardboard

Find vikings things is a VR game made for phones, where you have to find different vikings objects in a nice vikings village. At the start place you can see all the objects you need to find, you can unlock some achievements and a video will popup to show you the achievement you unlocked, when you get near an object that you have to find, a little sound starts reproducing and it keeps getting louder while you get closer the object with 3d sounds.
